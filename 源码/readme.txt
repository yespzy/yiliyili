该文件夹是本站的根目录文件夹，存放个页面文件、js脚本、css样式以及素材。
主要包含的文件有：

文件夹类：
文件夹 css			主要用于存放网页css样式文件
文件夹 files			主要用于存放新闻图片
文件夹 framework		主要用于存放各类开发框架
文件夹 icons			主要用于存放头像图标
文件夹 images		主要用于存放网页背景图片
文件夹 img			主要用于存放网页内容图片
文件夹 js			主要用于存放网页JavaScript代码文件

文件类：
about.html			“关于我们”页面的布局文档
backstage.php		“新闻后台管理”页面的php文件

global.php			网站连接数据库的php文件

hospital_example	医院数据的php文件
index.php			网站首页的php文件

login.php			登录页面的php文件
news.php			新闻页面的php文件
